# Adding variation data to our config

## CaeNDR resource

CaeNDR is the "The Caenorhabditis Natural Diversity Resource"

There is whole genome (short read) sequencing for hundreds of strains of C.
elegans, C.briggsae, and C.tropicalis

They publish VCF files and BAM files on their website

To load this resource into jbrowse we can run a couple simple commands

```bash
## soft-filtered variants
jbrowse add-track https://storage.googleapis.com/caendr-site-public-bucket/dataset_release/c_elegans/20231213/variation/WI.20231213.soft-filter.vcf.gz
jbrowse add-track https://storage.googleapis.com/caendr-site-public-bucket/dataset_release/c_elegans/20231213/variation/WI.20231213.soft-filter.isotype.vcf.gz

## hard-filtered variants
jbrowse add-track https://storage.googleapis.com/caendr-site-public-bucket/dataset_release/c_elegans/20231213/variation/WI.20231213.hard-filter.vcf.gz
jbrowse add-track https://storage.googleapis.com/caendr-site-public-bucket/dataset_release/c_elegans/20231213/variation/WI.20231213.hard-filter.isotype.vcf.gz

## imputed variants
jbrowse add-track https://storage.googleapis.com/caendr-site-public-bucket/dataset_release/c_elegans/20231213/variation/WI.20231213.impute.isotype.vcf.gz


## transposons
jbrowse add-track https://storage.googleapis.com/caendr-site-public-bucket/dataset_release/c_elegans/20231213/20231213_c_elegans_transposon_calls.bed
```
